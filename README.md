# Project: แอพรายรับรายจ่าย (Expense Tracker)

# เป้าหมาย:
สร้าง Web App สำหรับบันทึกรายรับรายจ่ายประจำวัน และสรุปยอดเงินคงเหลือ

# สิ่งที่ต้องการ (Requirements):
1. สร้างหน้าจอสำหรับกรอกข้อมูล: วันที่, รายการ, ประเภท (รับ/จ่าย), จำนวนเงิน
2. แสดงรายการย้อนหลังที่บันทึกไปแล้ว
3. คำนวณยอดรวมรายรับ, ยอดรวมรายจ่าย และเงินคงเหลือสุทธิ แสดงให้เห็นชัดเจน
4. ใช้เทคโนโลยี: React.js และ Ant Design โดยสามารถใช้อะไรก็ได้ที่ทำให้ใช้งานง่ายและทันสมัย
5. ออกแบบ UI ให้ดูทันสมัย ใช้งานง่าย บนมือถือ และ Desktop
6. ใช้ MongoDB ในการเก็บข้อมูล
7. สามารถแก้ไข และลบรายการได้
8. มีระบบ Login และ Register
9. มีระบบ Logout
10. มีระบบ Reset Password
11. มีระบบยืนยันตัวตนผ่าน Email
12. มีระบบยืนยันตัวตนผ่าน SMS
13. มีระบบยืนยันตัวตนผ่าน OTP
14. อยากได้หน้าจอ Dashboard ที่แสดงข้อมูลสรุปรายรับรายจ่าย
15. อยากได้หน้าที่ดูในรูปแบบปฏิทิน
